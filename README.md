# BaiduMap-iOS
使用百度地图API完成相关需求的iOS App Demo

## 主要功能：
1.全屏展示地图
2.获取当前定位信息
3.获取当前定位信息经纬度
4.移动地图，获取指针所指位置信息
5.通过地点名称搜索到相应地图位置，并跳转到对应位置
6.在App中展示地点导航路径
7.可跳转到第三方（百度、高德)导航软件进行导航

主界面：
